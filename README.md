# Quiz Game - React App

A simple quiz game built with React as a portfolio project.

## About

The user can select a category, difficulty level, and number of questions.
After each answer, feedback is displayed.
At the end of the quiz, the final score is shown, with options to reset or play again.

## Features

- Select category, difficulty, and number of questions
- Instant feedback after each answer
- Score tracking
- Play again or reset game

## Technologies

* HTML
* SCSS
* Javascript
* React
* Vite
* ESLint
